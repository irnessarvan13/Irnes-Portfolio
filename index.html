Background-Generator

Personal Project

For this project, I incorporated HTML, CSS, and JavaScript. This project's purpose was to have the user be able to choose what colors they wanted the background to be and it would change immediately on the website. When the user was satisfied with his linear-gradient background, it would give them the RGB number of the color they picked. This was also my first time using event listeners in JavaScript. I had a fun time making this project because while it is meant for people to find what RGB color they liked, I also come back to this website periodically and find a color that I like.
